# Quickstart: Alpha Content and External Readiness

## Content Audit

1. List included alpha spirits, structures, materials, and Codex chains.
2. Confirm each included item has a visible purpose.
3. Confirm each included item saves/loads.
4. Confirm deferred content is hidden, gated, or clearly unavailable.
5. Confirm first ritual, Red Fox, Meadow dwelling, upgraded Fox Den migration/Red-Fox-only double Satori generation, Dew Bowl, Wind Chime, Mist Stag, Ku Seed, Void separation, Chi+Ku calm-water island, Suijin invitation, and release-shell screens have no placeholder art, audio, icon, or UI assets.
6. Confirm non-primary placeholder assets, if any, are hidden, gated, or clearly outside the intended tester route.

## Tester Handoff

Prepare:

1. Menu-visible build version in `0.x.y-alpha+<build_id>` format.
2. Web alpha instructions.
3. Android install instructions.
4. Known issues.
5. What to test.
6. What is out of scope.
7. Feedback questions.

## Final Manual Playtest

Run one fresh-save playthrough on Web and one on Android through Suijin invitation before inviting testers.

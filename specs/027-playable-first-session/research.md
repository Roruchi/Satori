# Research: Playable First Session

## Decision: Keep the first-session slice narrow

**Rationale**: The alpha needs one complete learnable loop before content breadth. Meadow, Living Wood, Red Fox, and Warm Hollow are the smallest useful vertical slice.

## Decision: Use Codex and HUD hints sparingly

**Rationale**: The Codex should teach through hints, but the first 10 minutes cannot dead-end. UI should point to the next action only where the player lacks context.

## Decision: Validate with both GUT and manual play

**Rationale**: Ritual resolution and inventory are deterministic enough for GUT. Scene readability and first-time comprehension require manual validation.

## Resolved Decisions

- Red Fox is the first alpha spirit.
- First Meadow uses a normal-play First Bloom pacing rule: the first Living Wood appears quickly with visible growth feedback and no debug-only grants. Later growth can use the standard pacing.

# Contract: First Session Acceptance

## Purpose

Defines the minimum fresh-save first-session path for alpha.

## Contract

From a fresh save, the player can:

1. Start the game from title.
2. Create Meadow Seed from Wind/Fu.
3. Plant Meadow.
4. Grow or reach material-producing Meadow.
5. Harvest Living Wood.
6. Encounter or invite Red Fox.
7. Shape Warm Hollow from Living Wood and Fire Essence.
8. Place Warm Hollow as a valid Meadow dwelling.
9. Red Fox is automatically housed in that dwelling.
10. Save and reload without losing this state.

## Validation

- Focused GUT coverage for deterministic systems.
- Manual first-session script recorded in `quickstart.md`.

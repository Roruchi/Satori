# Specs Archive

This directory keeps superseded or historical Speckit feature specs.

Archived specs are not deleted because they explain how the project arrived at
its current direction. Active planning should use the numbered specs at the root
of `specs/` plus `specs/master/`.

Archive sets:

- `2026-06-ritual-material-migration/`: specs superseded by the ritual,
  materials, spirit-assistant and component direction.

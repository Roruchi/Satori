# Contract: First Island Loop Acceptance

## Purpose

Defines the first island behavior required before Ku/endgame work is meaningful.

## Contract

The first island must support:

- visible spirit housing or unrest state,
- upgraded Fox Den as the explicit first housing improvement,
- automatic Red Fox migration to placed Fox Den,
- double Satori generation for Red Fox only while Red Fox is in Fox Den,
- understandable Satori pressure and recovery,
- Dew Bowl as a storage/soothing helper,
- Wind Chime as an invitation/harvest helper,
- duplicate ritual rejection,
- invalid project feedback that does not start irreversible work,
- save/load preservation of island loop state, including Fox Den migration and Red-Fox-only double Satori generation.

## Validation

- GUT coverage for spirit, Satori, ritual, and structure rules.
- Manual first-island script recorded in `quickstart.md`.

# Ku Recipes Reference

This document lists the newly added Ku pair recipes for feature **016-add-ku-aether**.

| Recipe ID | Elements | Resulting Biome | Seed Display Name |
|---|---|---|---|
| `recipe_chi_ku` | Chi + Ku | `SACRED_STONE` | Sacred Stone Seed |
| `recipe_sui_ku` | Sui + Ku | `VEIL_MARSH` | Veil Marsh Seed |

- Solo Ku and undefined Ku combinations remain invalid.

# Discovery Audio Stingers

Discovery stingers are deferred for the closed alpha. Runtime discovery feedback is visual, while ambient and spirit audio are synthesized procedurally by `SoundscapeEngine`.

## Notes

- No authored discovery `.ogg` files are required for the alpha route.
- `DiscoveryAudioPlayer.AUDIO_MAP` intentionally stays empty until final-enough stingers are added.
- Missing stingers do not block visual discovery notifications.
